How to initialize a pointer to nullptr

When a pointer is initialized to 0(or nullptr), it has pointed to null.
That means it hasn't allocated any contiguous memory region to store a data set yet.
Therefore, don't expect it has an array in it and the value is set to '\0.'
Note: 0 interpreted as nullptr on VS compiler

char* str{};
char* str1{0};
char* str2(nullptr);
char* str3(0);
cout << (str==nullptr) << (str1==nullptr) << (str2==nullptr) << str3==nullptr);
cout << (str==0) << (str1==0) << (str2==0) << (str3==0);

All of them are True
