How cin extracts data from input stream

for integer type
If invalid integer(non-numeric characters) is inputted into the input stream, the state of istream cin turns to fail.
Also, if floating point number is inputted, it truncates decimals.

int value;
cin >> value;
if (cin.fail()) cout << "cin failed - invalid value: " << value << endl;
else cout << "valid value: " << value << endl;

cin.fail() can check if user input is valid
